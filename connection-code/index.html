The read on Phase.

Four questions. The five Phases of a Connection Code arc are Profile, Connection, Courtship, Commitment, Ongoing. The Engine reads where you arrive.

Stage Four · Signal

The read on Signal.

Three questions. The five Signals of the Connection Code are Intent, Position, Open, Build, Progress. Intent and Position read upstream; Open, Build, and Progress are the read in motion.

The frame

Method, not luck.

Most failed connections trace to a missed Signal. The Search arm names the architecture — the Phases connections move through, the Signals each Phase produces, and the Intents that calibrate why the work is being done at all.

The user does not need to be a methodology practitioner to benefit from the framework. The architecture surfaces what the body and the early-dating cycle were already doing — the Search arm makes the patterns visible so the user can act from chosen ground rather than from spike-reactivity.

Studio AI activates the Search

The methodology, applied to the Search context.

Studio AI surfaces the Search-context tools that operate the methodology in real situations. Coach AI for the moment-to-moment questions. Script Builder for the calibrated invitation, the hard conversation, the chosen reply. Suitability for the read-fit-before-time-commitment work the framework asks the user to do before deeper engagement.

  • Coach AI Ask a methodology-grounded question about the Search context; receive an answer cited to the Playbook chapter that grounds the response. The Coach surfaces the framework; the user does the work.
  • Script Builder Invocation of the methodology's communication frameworks (Active Listening, Naming-Observation-Feeling-Request, Boundary Conversation, Repair Attempt, others) calibrated to the user's Reading location and the situation pattern.
  • Suitability Methodology-grounded read of fit between the user and a potential connection. Pattern recognition without prediction; descriptive without prescription.

The Studio AI tools are activation, not substitution. The framework is what does the work; the tools surface the framework in real situations.
